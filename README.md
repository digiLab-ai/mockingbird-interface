## Requirements

-   [Python](https://www.python.org/downloads/)
-   [Poetry](https://python-poetry.org/docs/#installation)

## Quickstart

Change to the `interface` directory:

```bash
cd interface
```

Install the dependencies:

```bash
poetry install
```

## Testing

Run the tests:

```bash
poetry run pytest
```
